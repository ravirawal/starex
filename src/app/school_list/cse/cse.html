Foundational Programs

Starex University's School of Computer Science is dedicated to nurturing innovative minds who will drive the technological advancements of tomorrow. We offer a comprehensive curriculum that blends academic rigor with practical application, preparing our students to excel in the dynamic world of computing.

Bachelor of Computer Applications (BCA)

This three-year program offers a well-rounded curriculum encompassing programming, databases, software development, and emerging technologies like Artificial Intelligence and Machine Learning.

Bachelor of Computer Applications (BCA) with Specialization

Our BCA with specialization in Python and Azure for Data Cloud Integration equips students with in-demand skills for managing and analyzing large datasets in the cloud.

Advanced Studies

Master of Computer Applications (MCA)

This two-year program provides advanced knowledge in software development, data science, cybersecurity, and other specialized areas of computer applications.

MCA with Microsoft Specialization in Python and Azure for Data Cloud Integration

This specialization focuses on equipping students with skills in Python programming and Microsoft Azure for cloud-based data integration, preparing them for careers in cloud computing and data management.
